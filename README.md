# GPP Golang Demo

A simple project highlighting some key introductory concepts in Golang

### Get It
```go
go get -u github.com/eunanibus/gpp-demo-project
```

### Run it

To run in a docker instance
```go
make run
```

or to run locally

```go
go run cmd/gpp-demo-project/main.go
```

### Test it
```go
make test
```
